# 痞客邦文章備份腳本

這個腳本使用 puppteer 操作瀏覽器進行爬蟲。(單純為了小腳本開發快一點)

# 開始使用

1. clone 這個專案
2. 在終端機中輸入 `npm i` (因 puppteer 包含了一個瀏覽器，所以下載需要一段時間)
3. 修改腳本：在 [index.js](./index.js) 中的 11 12 行，改成你的名字，以及主頁下面總共有幾個分頁
4. `npm run start` 即可等待他慢慢把你的文章抓下來
